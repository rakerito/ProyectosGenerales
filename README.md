# Proyectos Generales - Aplicaciones Web Orientadas a Servicios

## Descripción
Repositorio con proyectos realizados en la materia de **Aplicaciones Web Orientadas a Servicios**.  
Incluye ejemplos prácticos desarrollados en **Python**, enfocados en el consumo y creación de servicios web.

## Contenido
- **Proyecto 1**: Ejercicio inicial de prueba con servicios web.
- (Puedes ir agregando más proyectos conforme avances).

## Objetivo académico
Estos proyectos fueron diseñados para:
- Practicar el desarrollo de aplicaciones web con enfoque en servicios.
- Comprender la interacción entre cliente y servidor.
- Aplicar conceptos de programación en Python orientados a servicios.
